# KIN Pro Teleprompter
KIN Pro Teleprompter is a lightweight, browser-based teleprompter designed for creators,
professionals, and thought leaders who record long-form video content.

It provides a smooth, professional reading experience with powerful, modern recording features, all with zero setup.

## Key Features

### Smart Script Parsing
* Paste any script, blog post, or notes.
* Automatically splits content into readable segments.
* **New:** Robust segmentation logic prevents splitting at common abbreviations (e.g., 'Mr.', 'Dr.', 'etc.') for cleaner delivery.
* Optional splitting at commas for finer delivery control.
* Preview exactly how the script will appear before recording.

### Professional Teleprompter Experience
* Centered scrolling for ideal focus.
* Clear visual emphasis on the current line.
* Past and future lines fade naturally.
* Adjustable scroll speed (milliseconds per segment).
* **New:** Real-time adjustable font size (2rem to 6rem) for optimal reading distance.

### Built-In Video Recording
* Uses your browserâ€™s camera and microphone.
* Select specific camera and microphone sources.
* Live camera preview with mirrored/flipped controls.
* One-click recording with a 3-second countdown timer.
* Auto-stop recording at end of script.
* Download recordings in .webm format.
* **New:** Smooth, non-blocking error notifications replace jarring browser alerts for media access issues.

### User Experience
* Camera Orientation Controls (Mirror mode and Vertical Flip) for specialized setups.
* Zero Setup (No installation, no account required, runs entirely in the browser).

## Privacy
* No data stored.
* All processing happens locally on your device.

## Tech Stack
* HTML5
* CSS
* Vanilla JavaScript
* MediaRecorder API
